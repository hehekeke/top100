什么是TOP100？

TOP100SUMMIT是科技界一年一度的案例研究盛会，每年甄选有代表的100个技术创新/研发管理案例，旨在揭幕100件案例背后的思考、长尾价值，为 听众提炼最佳学习路径（或看点），帮助他人的项目或团队获得启示、成长，正如”他山之石，可以攻玉”。注 大会由15位垂直专题领域的业界专家担当出品人，负责专题案例评审、甄选，100件案例由五个专题构成：产品创新/用户体验、团队管理/组织发展 、架构设计/技术战略、设计匠艺/流程再造、测试管理/质量平台。

组委会怎么构成的？如何保证案例的品质呢？

1. 由知名培训咨询公司麦思博（msup）主办；
2. 来自行业媒体、认证机构、软件协会、业界企业等35家理事单位共同推动案例征集；
3. 由15位行业专家组成的专家团担任大会出品人,甄选有学习参考价值的案例

组委会海选案例后有哪些规划或分类？

1. 产品创新/用户体验（用户体验、产品创新、商业模式……）
2. 团队管理/组织发展（绩效管理、团队管理、组织变革转型……）
3. 架构设计/技术战略（云计算、大数据、架构设计……）
4. 设计匠艺/流程再造（精益开发、代码设计、流程、过程改进……）
5. 测试管理/质量平台（质量体系、自动化测试、运行保障……）

入选案例有什么标准？一个让听众期待的教学案例有哪些关键指标？

1. 来自科技领域的一线研发管理实践或产品创新案例；
2. 入选者的产品或项目案例（再或从事的工程实践）具有借鉴、研究学习的长尾价值，可以帮助他人的项目或团队获得启示、成长；
3. 入选者是一位热爱创新、敏于思考、善于总结经验的“教练”；
4. 谢绝广告性、赞助性演讲，不参与夹杂产品宣传、个人包装色彩的案例；
5. 入选流程：案例模板提交——初选意见——复选评审——大会现场解读。

入选后有费用吗？

1. 分享者不用缴纳任何费用,大会旨在案例研究、交流学习为目的
2. 分享者担负往返北京国际会议中心所需的机票、住宿等差旅费用；（对于经济困难的分享者，此费用可通过申请《全球软件案例研究基金》来获得资助）
案例入选者可以获得什么？
    1. 分享者或所在团队入选《TOP 100 Case Studies》奖章，并受邀以“理事单位”的身份出席峰会；同时，理事单位独享10张五折购票券； 优秀案例文章入选《全球软件案例研究年鉴》图书，部分杰出文章还有机会获得业界权威杂志刊登、媒体报道、副刊专辑；
    2. 分享者获得VIP通行证，全程会议以及精彩讲师聚餐（四天午餐、晚餐）；
    3. 分享者成为msup教练俱乐部成员，参加不定期组织的企业互访、聚餐；
    4. 获得一天《金牌教练》课程，发现自己、更精彩的讲演您的案例；
    5. 获得更多行业交流圈子，增加媒体曝光机会；
